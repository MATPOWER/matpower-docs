mp.dme_bus_opf

class mp.dme_bus_opf

Bases: mp.dme_bus, mp.dme_shared_opf

mp.dme_bus_opf - Data model element for bus for OPF.

To parent class mp.dme_bus, adds shadow prices on power balance and voltage magnitude limits, and pretty-printing for lim sections.

Adds the following columns in the main data table, found in the tab property:

Name

Type

Description

lam_p

double

active power nodal price, i.e. shadow price on active power balance constraint (u/MW) [1]

lam_q

double

reactive power nodal price, i.e. shadow price on reactive power balance constraint (u/MVAr) [1]

mu_vm_lb

double

shadow price on voltage magnitude lower bound (u/p.u.) [1]

mu_vm_ub

double

shadow price on voltage magnitude upper bound (u/p.u.) [1]
[1]

Here u denotes the units of the objective function, e.g. USD.
