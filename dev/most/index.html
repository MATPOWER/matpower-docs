MOST User’s Manual

Note

The new web-based version of the MOST User’s Manual is not yet available. Please, continue to use the PDF version of the MOST User's Manual for now, along with the new reference documentation below.

Reference

The primary sources of documentation for MOST, aside from the legacy PDF MOST User's Manual, are this Reference documentation and the built-in help command. As with the built-in functions and toolbox routines in MATLAB and GNU Octave, you can type help followed by the name of a command or M-file to get help on that particular function.
