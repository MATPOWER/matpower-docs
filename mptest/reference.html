Reference

Testing Functions

Use these functions in implementing your own tests.

  • t_begin
  • t_end
  • t_file_match
  • t_is
  • t_ok
  • t_run_tests
  • t_skip
  • t_str_match

Other Functions

Use these functions to test for availability and version information for optional functionality and to check the version of the installed MP-Test.

  • have_feature
  • mptestver

Tests of MP-Test

These functions test that MP-Test is installed and functioning as expected.

  • test_mptest
  • t_have_feature
  • t_test_fcns

Private Functions

The following are private functions that implement detection of specific optional functionality. They are not intended to be called directly, but rather are used to extend the capabilities of have_feature() (see above).

  • have_feature_matlab
  • have_feature_octave
